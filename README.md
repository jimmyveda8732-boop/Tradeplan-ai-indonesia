# TradePlan AI Indonesia

Versi UI 1.0 untuk aplikasi analisis screenshot saham dan trading plan harian Indonesia.

## Fitur versi awal

- Dashboard responsif untuk HP, tablet, dan komputer
- Form upload maksimal 6 screenshot
- Pilihan jenis screenshot
- Input modal dan profil risiko
- Tampilan contoh hasil BUY, HOLD, dan WATCHLIST
- Dasar PWA melalui manifest
- Siap dipublikasikan ke Vercel

## Menjalankan secara lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Tahap berikutnya

1. Deploy ke Vercel
2. Hubungkan Supabase
3. Tambahkan login
4. Simpan screenshot
5. Hubungkan AI pembaca gambar
6. Tambahkan Yahoo Finance dan berita
7. Aktifkan generator infografis

## Disclaimer

Aplikasi ini bukan ajakan membeli atau menjual saham.
